# Personal Portfolio - Barra Akmal

A personal portfolio website highlighting academic achievements, school experiences, and coding projects.

## 🚀 Features
- **Responsive Design**: Optimized for all devices (Desktop, Tablet, Mobile).
- **Dark Mode Support**: Seamless toggle between light and dark themes.
- **Academic Roadmap**: Detailed future plans and university aspirations.
- **Experience Timeline**: Documentation of school activities like PROPIL, LDK, and EXPERT.
- **Coding Portfolio**: Integration with GitHub repositories.

## 📁 Project Structure
- `index.html`: Main structure and content.
- `style.css`: Custom animations and layout styles.
- `script.js`: Navigation logic and theme toggling.
- `*.png / *.jpeg`: Project and experience documentation.
