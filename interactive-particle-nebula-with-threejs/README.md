# Interactive Particle Nebula with Three.js

A Pen created on CodePen.

Original URL: [https://codepen.io/oftaptycode/pen/ogzyGXw](https://codepen.io/oftaptycode/pen/ogzyGXw).

interactive particle nebula created with Three.js and WebGL. This simulation features tens of thousands of particles that form a colorful, rotating galaxy.

Interactive Mouse Trail: Move your mouse across the screen to gently push the particles away, creating a beautiful disturbance in the nebula.

Click to Ripple: Click anywhere to create an expanding ripple effect that pushes particles and makes them glow.

Bloom Effect: A post-processing bloom effect (UnrealBloomPass) gives the particles a vibrant, ethereal glow.

Dynamic Physics: Particles have their own simple physics, including velocity, damping, and a force that pulls them back to their original position, creating a fluid and organic feel.

This pen is a great example of using BufferGeometry for performance with a large number of objects and implementing interactive effects in a 3D scene.
